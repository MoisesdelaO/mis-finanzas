🔗 Conexión Google Sheets
  1. Crea una hoja en Google Sheets llamada "Mis Finanzas"
  2. Ve a Extensiones → Apps Script
  3. Pega el contenido de apps_script.gs y guarda
  4. Implementar → Nueva implementación → Aplicación web → Acceso: Cualquier persona
  5. Copia la URL y pégala abajo
📥 Importar datos

Carga un archivo JSON exportado previamente desde esta app. Tus datos actuales serán reemplazados.

Para limpiar datos en todos los dispositivos: borra el contenido de tu Google Sheet manualmente, luego usa "Borrar todos los datos" aquí. Al abrir la app en otros dispositivos, sincronizarán automáticamente con el Sheet vacío.
